About Me

Hi, my name is Matt. I’m a front-end and back-end developer with over ten years of experience. I have a Bachelor’s degree in Computer Science and a minor in Radio and Television Broadcasting.

I currently work for Sharefaith.com, where I am the senior developer for church websites. Over the past two years, I've launched Sharefaith's church website template business, and brought it up to its current 3500+ website customers. My responsibilities include architecting the website master plan and future projects for the department, as well as overseeing development.

I designed and coded a project called Sidekick that transforms a standard WordPress site into a front-end editable administration system. Menus are sortable, graphics are drag-and-drop, the design is real-time swappable, banners can be edited directly, and much more. Feel free to give it a test run.

When I’m not working, I love spending time with my wonderful family. I have a three year old lovely girl, and a 2 year old spunky boy. My wife, Amy, is expecting another baby in October.

Sidekick

Sidekick is an admin tool for managing the user’s website. It allows the user to change their template between all the different designs we offer with a simple click. Menus can be changed by simple drag and drop. All pre-designed banners and sidebars can be downloaded used by simply dragging them from the media drawer to the website.

Try a demo using the link above.
Sidekick Image Editor

This product came out of my experimentation with HTML5 canvas. It enables the user to add one of our pre-designed background images and then add text using any font on the user’s computer. The image project is saved into the database for future editing. Give it a try using the website demos.

Try a demo using the link above.
Sharefaith Updater

All of our users get instant updates whenever we release new features or bug fixes. The template checks with my updater system to see that their user account is still active. If so, their site auto-updates to the new template.

Sharefaith Kids
